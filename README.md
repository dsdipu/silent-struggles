# 🌫️ Silent Struggles
Silent Struggles is a mental health awareness landing page built as a practical front-end project.  
This project focuses on clean UI design, structured layout, and Tailwind CSS implementation to create a compassionate space that addresses mental health topics with sensitivity and care.

🔗 Live Demo: [View](https://dsdipu.github.io/silent-struggles/)
🔗 Repo Link: [View](https://github.com/dsdipu/silent-struggles)

---

## 📌 Overview

Silent Struggles is a static mental health awareness landing page designed to create a safe and empathetic digital space for those facing mental health challenges. This project was developed for practical learning purposes to strengthen front-end development skills, particularly layout structuring and utility-first CSS styling using Tailwind CSS. The website currently focuses on UI design only and does not include backend functionality or JavaScript-based interactivity, though visual storytelling plays a key role in conveying its message.

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS (CLI)
- Font Awesome

## ✨ Features

- Sticky Header
- Hero Section with Emotional Impact
- Resource/Help Section
- Footer Section
- Hover Effects
- Clean and Structured Layout
- Utility-first CSS Styling
- Visual Storytelling through Imagery

## ⚙️ Installation & Setup

To run the project locally with Tailwind CSS CLI:

```bash
npx @tailwindcss/cli -i src\input.css -o src\output.css --watch
```
Then open index.html in your browser.
